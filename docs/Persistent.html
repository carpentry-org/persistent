Persistent

Macro-generated persistent data structures backed by Rc.

define-deque

macro

Macro 

                        (define-deque name value-type)

Generate a persistent deque backed by two shared lists.

Example:

(Persistent.define-deque IntDeque Int)

This generates:

  • type IntDeque
  • module IntDeque with deque operations
  • internal list type used as front/rear storage

define-hash-map

macro

Macro 

                        (define-hash-map name key-type value-type)

Generate a persistent hash map using trie-indexed hash buckets.

Example:

(Persistent.define-hash-map StrIntMap String Int)
